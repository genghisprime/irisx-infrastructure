GETTING STARTED:
---------------

1. Choose an example:
   cd irisx-examples/[example-name]

2. Install dependencies:
   npm install

3. Configure environment:
   cp .env.example .env
   # Edit .env with your IRISX credentials

4. Run the example:
   npm start

All examples are production-ready and follow industry best practices!
