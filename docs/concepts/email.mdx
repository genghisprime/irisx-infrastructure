## Overview

IRISX provides reliable email delivery via SendGrid and AWS SES with tracking, templates, and bounce handling.

## Sending Email

### Basic Email

```javascript
const email = await irisx.email.send({
  to: 'customer@example.com',
  from: 'noreply@yourcompany.com',
  subject: 'Welcome to Our Service',
  body_text: 'Thank you for signing up!',
  body_html: '<h1>Welcome!</h1><p>Thank you for signing up!</p>'
});
```

### With Attachments

```javascript
const email = await irisx.email.send({
  to: 'customer@example.com',
  from: 'noreply@yourcompany.com',
  subject: 'Your Invoice',
  body_html: '<p>Please find your invoice attached.</p>',
  attachments: [
    {
      filename: 'invoice.pdf',
      content: base64Content,
      content_type: 'application/pdf'
    }
  ]
});
```

## Email Templates

```javascript
// Create template
const template = await irisx.email.createTemplate({
  name: 'password_reset',
  subject: 'Reset Your Password',
  body_html: '<p>Hi {{name}}, click here to reset: {{reset_link}}</p>'
});

// Send with template
const email = await irisx.email.sendTemplate({
  to: 'user@example.com',
  template_slug: 'password_reset',
  variables: {
    name: 'John',
    reset_link: 'https://yourapp.com/reset/abc123'
  }
});
```

## Tracking

Track opens and clicks:

```javascript
const email = await irisx.email.send({
  to: 'customer@example.com',
  from: 'noreply@yourcompany.com',
  subject: 'Check out our new features',
  body_html: '<p>Visit <a href="https://yourapp.com">our site</a></p>',
  track_opens: true,
  track_clicks: true
});

// Get stats later
const stats = await irisx.email.getStats(email.id);
console.log(`Opened: ${stats.opened}`);
console.log(`Clicked: ${stats.clicked}`);
```

## Email States

- **queued:** Accepted and queued
- **sending:** Being sent
- **sent:** Delivered to recipient's mail server
- **delivered:** Confirmed delivery
- **opened:** Recipient opened email
- **clicked:** Recipient clicked link
- **bounced:** Hard or soft bounce
- **complained:** Marked as spam
- **failed:** Send failed
