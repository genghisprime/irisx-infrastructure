## Overview

Register a new tenant account and receive authentication tokens. This endpoint creates both a tenant (company) account and an admin user in a single request.

<Note>
  This endpoint does not require authentication. New accounts receive $100 in free credits during beta.
</Note>

## Request Body

<ParamField body="company_name" type="string" required>
  Company or organization name
</ParamField>

<ParamField body="email" type="string" required>
  Admin user email address
</ParamField>

<ParamField body="password" type="string" required>
  Password for the admin user (minimum 8 characters)
</ParamField>

<ParamField body="first_name" type="string" required>
  Admin user's first name
</ParamField>

<ParamField body="last_name" type="string" required>
  Admin user's last name
</ParamField>

## Response

<ResponseField name="token" type="string">
  JWT access token for immediate API access (expires in 24 hours)
</ResponseField>

<ResponseField name="refresh_token" type="string">
  Refresh token for obtaining new access tokens (expires in 30 days)
</ResponseField>

<ResponseField name="user" type="object">
  Admin user object

  <Expandable title="User properties">

    <ResponseField name="tenant_id" type="integer">

    <ResponseField name="email" type="string">
      User email address
    </ResponseField>
    <ResponseField name="first_name" type="string">
      First name

    <ResponseField name="last_name" type="string">

    <ResponseField name="role" type="string">

```javascript Node.js
const axios = require('axios');

async function register() {
  try {
    const response = await axios.post(
      'https://api.useiris.com/v1/auth/register',
      {
        company_name: 'Acme Corporation',
        email: 'admin@acme.com',
        password: 'securepassword123',
        first_name: 'John',
        last_name: 'Doe'
      }
    );

    console.log('Registration successful!');
    console.log('Token:', response.data.token);
    console.log('API Key:', response.data.tenant.api_key);

    // Store tokens securely
    localStorage.setItem('token', response.data.token);
    localStorage.setItem('refresh_token', response.data.refresh_token);

  } catch (error) {
    console.error('Registration failed:', error.response?.data);
  }
}
```

```python Python
import requests

def register():
    response = requests.post(
        'https://api.useiris.com/v1/auth/register',
        json={
            'company_name': 'Acme Corporation',
            'email': 'admin@acme.com',
            'password': 'securepassword123',
            'first_name': 'John',
            'last_name': 'Doe'
        }
    )

    if response.status_code == 201:
        data = response.json()
        print('Registration successful!')
        print(f"Token: {data['token']}")
        print(f"API Key: {data['tenant']['api_key']}")

        # Store tokens securely
        return data
    else:
        print(f"Registration failed: {response.json()}")
```

## Error Codes

| Code | Error | Description |
|------|-------|-------------|
| 400 | VALIDATION_ERROR | Invalid or missing required fields |
| 400 | DUPLICATE_EMAIL | Email address already registered |
| 400 | WEAK_PASSWORD | Password doesn't meet minimum requirements |
| 500 | INTERNAL_ERROR | Server error during registration |

## Security Notes

<Warning>
  **Store credentials securely!** The API key is only returned once during registration. Store it in a secure location (environment variables, secrets manager).
</Warning>

<Tip>
  Use the JWT token for user-facing applications (customer portal, agent desktop) and the API key for server-to-server communication.
</Tip>
