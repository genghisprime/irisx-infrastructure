## Overview

Retrieve a list of all webhook endpoints configured for your account.

## Authentication

<ParamField header="X-API-Key" type="string">
  API key for server-to-server authentication
</ParamField>

<ParamField header="Authorization" type="string">
  Bearer token for user authentication
</ParamField>

## Response

<ResponseField name="webhooks" type="array">
  Array of webhook endpoint objects
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.useiris.com/v1/webhooks/endpoints \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript Node.js
const axios = require('axios');

async function listWebhooks() {
  const response = await axios.get(
    'https://api.useiris.com/v1/webhooks/endpoints',
    {
      headers: { 'X-API-Key': 'YOUR_API_KEY' }
    }
  );

  console.log('Found', response.data.webhooks.length, 'webhooks');
  return response.data.webhooks;
}
```

```python Python
import requests

def list_webhooks():
    response = requests.get(
        'https://api.useiris.com/v1/webhooks/endpoints',
        headers={'X-API-Key': 'YOUR_API_KEY'}
    )

    if response.status_code == 200:
        webhooks = response.json()['webhooks']
        print(f"Found {len(webhooks)} webhooks")
        return webhooks
```
