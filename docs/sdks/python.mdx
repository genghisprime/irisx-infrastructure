## Installation

Install the IRISX SDK via pip:

```bash
pip install irisx-sdk
```

Or with Poetry:

```bash
poetry add irisx-sdk
```

## Quick Start

```python
from irisx import IRISX

# Initialize with API key
irisx = IRISX(api_key='your_api_key_here')

# Make your first call
def make_call():
    call = irisx.calls.create(
        to='+15551234567',
        from_='+15559876543',
        record=True
    )
    print(f"Call initiated: {call.uuid}")

make_call()
```

### Verify Webhook Signature

```python
import hmac
import hashlib
import json

def verify_webhook(payload: dict, signature: str, secret: str) -> bool:
    """Verify webhook signature using HMAC-SHA256"""
    expected_signature = hmac.new(
        secret.encode(),
        json.dumps(payload).encode(),
        hashlib.sha256
    ).hexdigest()

    return hmac.compare_digest(signature, expected_signature)

# With Flask
from flask import Flask, request

app = Flask(__name__)

@app.route('/webhooks/irisx', methods=['POST'])
```

## Error Handling

```python
from irisx.exceptions import IRISXError, RateLimitError, ValidationError

try:
    call = irisx.calls.create(
        to='+15551234567',
        from_='+15559876543'
    )
except RateLimitError as e:
    print(f'Rate limited - retry after: {e.retry_after}s')
    # Wait and retry
    import time
    time.sleep(e.retry_after)
except ValidationError as e:
    print(f'Validation error: {e.message}')
    print(f'Details: {e.details}')
except IRISXError as e:
    print(f'API Error: {e.message}')
    if e.code == 'INSUFFICIENT_FUNDS':
        print('Low balance - add credits')
except Exception as e:
    print(f'Unexpected error: {e}')
```

### Error Types

The SDK provides specific error classes:

```python
from irisx.exceptions import (
    IRISXError,
    RateLimitError,
    ValidationError,
    AuthenticationError,
    NotFoundError
)

try:
    number = irisx.phone_numbers.get('invalid_id')
except NotFoundError:
    print('Phone number not found')
except AuthenticationError:
    print('Authentication failed - check your API key')
except Exception as e:
    print(f'Error: {e}')
```

## Advanced Features

### Automatic Retry with Exponential Backoff

```python
irisx = IRISX(
    api_key='your_api_key_here',
    retry=True,
    max_retries=3,
    retry_delay=1.0,  # 1 second
    max_retry_delay=30.0  # 30 seconds
)
```

### Custom Timeout

```python
irisx = IRISX(
    api_key='your_api_key_here',
    timeout=60  # 60 seconds
)
```

```python
logging.basicConfig(level=logging.DEBUG)
```
