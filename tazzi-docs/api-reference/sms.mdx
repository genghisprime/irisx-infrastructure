# SMS API

Send and manage SMS/MMS messages.

## Send SMS

`POST /v1/sms/send`

Send an SMS or MMS message.

## Get Message

`GET /v1/sms/{id}`

Retrieve message details.

## List Messages

`GET /v1/sms`

List all messages.
