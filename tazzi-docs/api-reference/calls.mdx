# Calls API

Create and manage voice calls. See the OpenAPI specification for full details.

## Create Call

`POST /v1/calls`

Make an outbound call.

## Get Call

`GET /v1/calls/{id}`

Retrieve call details.

## List Calls

`GET /v1/calls`

List all calls with filtering.
